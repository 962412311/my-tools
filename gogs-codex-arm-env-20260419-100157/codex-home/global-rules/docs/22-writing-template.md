# 22-writing-template

## 目的
本文件定义常见文档的推荐模板，目标是降低写作成本并提高一致性。

---

## 通用模板
适用于大多数正式文档：

```md
# 标题

## 概述
一句话说明文档要解决什么问题。

## 背景
为什么会有这份文档。

## 目标
这份文档服务什么目标。

## 范围
包含什么，不包含什么。

## 正文
按主题分节展开。

## 风险或待定项
有哪些尚未明确的点。

## 下一步
接下来应该做什么。
```

---

## README 模板

```md
# 项目名

## 项目简介
一句话说明项目用途。

## 快速开始
如何安装、启动、验证。

## 目录结构
各核心目录的职责。

## 常用命令
开发、测试、构建、部署相关命令。

## 约定
命名、规范、注意事项。

## 常见问题
容易踩坑的点。
```

## Spec 模板

```md
# 功能 / 方案名称

## 背景
## 目标
## 非目标
## 用户场景
## 方案说明
## 数据 / 接口影响
## 风险
## 验收标准
```

## Meeting Notes 模板

```md
# Meeting Notes - YYYY-MM-DD

## 会议主题
## 参与者
## 背景
## 讨论要点
## 结论
## 待办事项
## 风险 / 阻塞项
```

---

## SOP 模板

```md
# SOP - 主题

## 目的
## 适用范围
## 前置条件
## 操作步骤
## 验证方式
## 常见问题
## 回滚 / 兜底方案
```

---

## Decision Record 模板

```md
# Decision - YYYY-MM-DD - Topic

## 背景
## 决策内容
## 原因
## 备选方案
## 影响
## 后续动作
```

---

## 使用原则
- 能套模板时，不从空白开始写
- 同类文档统一模板
- 模板可以裁剪，但不要失去结构
- 文档首先服务理解与执行，不为形式而写
