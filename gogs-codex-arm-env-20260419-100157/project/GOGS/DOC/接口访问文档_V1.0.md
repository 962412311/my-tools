## 1. 接口概述

### 1.1 接口架构

系统采用 RESTful API 架构设计，基于 HTTP/HTTPS 协议进行通信。

- **后端服务**: Qt HTTP Server (QHttpServer)
- **前端框架**: Vue 3 + Axios
- **数据格式**: JSON
- **认证方式**: JWT (JSON Web Token)

### 1.2 基础信息

| 项目 | 开发环境 | 生产环境 |
|------|----------|----------|
| 基础URL | `http://localhost:8080/api` | `https://{server_ip}/api` |
| WebSocket | `ws://localhost:12345` | `wss://{server_ip}/ws` |
| 超时时间 | 10秒 | 10秒 |

### 1.3 接口模块划分

```
/api
├── /auth          # 认证授权模块
├── /users         # 用户管理模块
├── /history       # 历史记录模块
├── /system        # 系统状态模块
├── /inventory     # 库存管理模块
├── /config        # 配置管理模块
└── /material      # 物料操作模块
```

## 2. 通用规范

### 2.1 请求规范

#### 请求头 (Headers)

| 字段名 | 必填 | 说明 |
|--------|------|------|
| Content-Type | 是 | `application/json` |
| Authorization | 条件 | `Bearer {token}` (需要认证的接口) |
| Accept | 否 | `application/json` |

#### 请求方法

| 方法 | 用途 |
|------|------|
| GET | 获取资源 |
| POST | 创建资源 |
| PUT | 更新资源 |
| DELETE | 删除资源 |

### 2.2 响应规范

#### 成功响应格式

```json
{
  "success": true,
  "data": { ... },
  "message": "操作成功"
}
```

#### 错误响应格式

```json
{
  "success": false,
  "error": "错误描述",
  "code": 4001
}
```

#### 分页响应格式

```json
{
  "success": true,
  "data": {
    "records": [ ... ],
    "total": 100,
    "page": 1,
    "pageSize": 20,
    "totalPages": 5
  }
}
```

### 2.3 通用状态码

| HTTP状态码 | 含义 |
|------------|------|
| 200 | 请求成功 |
| 400 | 请求参数错误 |
| 401 | 未授权/Token无效 |
| 403 | 禁止访问/权限不足 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
| 503 | 服务不可用 |

## 3. 认证授权接口

### 3.1 用户登录

**接口地址**: `POST /api/auth/login`

**接口说明**: 用户登录获取访问令牌

**请求参数**:

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| username | string | 是 | 用户名 |
| password | string | 是 | 密码 (MD5加密) |
| captcha | string | 否 | 验证码 |

**请求示例**:

```json
{
  "username": "admin",
  "password": "e10adc3949ba59abbe56e057f20f883e",
  "captcha": "1234"
}
```

**响应参数**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| success | boolean | 是否成功 |
| token | object | Token信息 |
| token.access_token | string | 访问令牌 |
| token.refresh_token | string | 刷新令牌 |
| token.expires_in | number | 过期时间(秒) |
| user | object | 用户信息 |
| user.id | number | 用户ID |
| user.username | string | 用户名 |
| user.role | string | 角色 |
| user.real_name | string | 真实姓名 |

**响应示例**:

```json
{
  "success": true,
  "token": {
    "access_token": "eyJhbGciOiJIUzI1NiIs...",
    "refresh_token": "eyJhbGciOiJIUzI1NiIs...",
    "expires_in": 7200
  },
  "user": {
    "id": 1,
    "username": "admin",
    "role": "admin",
    "real_name": "管理员"
  }
}
```

**错误响应**:

```json
{
  "success": false,
  "message": "用户名或密码错误",
  "fail_reason": "INVALID_CREDENTIALS",
  "remaining_attempts": 3
}
```

## 5. 历史记录接口

### 5.1 查询操作记录

**接口地址**: `GET /api/history/operations`

**接口说明**: 分页查询物料操作历史记录

**请求参数 (Query)**:

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| page | number | 否 | 页码，默认1 |
| pageSize | number | 否 | 每页条数，默认20，最大100 |
| startTime | string | 否 | 开始时间(ISO 8601格式) |
| endTime | string | 否 | 结束时间(ISO 8601格式) |
| operationType | string | 否 | 操作类型: DECREASE/INCREASE |
| operationSource | string | 否 | 操作来源: GRAB/SCAN/MANUAL |
| pileId | string | 否 | 料堆ID |
| sortBy | string | 否 | 排序字段，默认created_at |
| sortOrder | string | 否 | 排序方向: asc/desc，默认desc |

**响应参数**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| records | array | 记录列表 |
| total | number | 总记录数 |
| page | number | 当前页码 |
| pageSize | number | 每页条数 |
| totalPages | number | 总页数 |
| todayStats | object | 今日统计 |

**响应示例**:

```json
{
  "success": true,
  "data": {
    "records": [
      {
        "id": 1001,
        "operationType": "DECREASE",
        "operationSource": "GRAB",
        "pileId": "P001",
        "pileName": "料堆A-001",
        "volumeBefore": 1250.50,
        "volumeAfter": 1180.25,
        "volumeChange": -70.25,
        "grabCount": 3,
        "remark": "自动抓料作业",
        "createdAt": "2026-03-15T10:30:00Z",
        "updatedAt": "2026-03-15T10:30:00Z"
      }
    ],
    "total": 156,
    "page": 1,
    "pageSize": 20,
    "totalPages": 8,
    "todayStats": {
      "total": 25,
      "decrease": 15,
      "increase": 10,
      "totalVolumeChange": 1250.5
    }
  }
}
```

### C. 接口调用示例

#### JavaScript/Axios 示例

```javascript
import axios from 'axios';

const api = axios.create({
  baseURL: 'http://localhost:8080/api',
  timeout: 10000,
  headers: { 'Content-Type': 'application/json' }
});

// 添加Token拦截器
api.interceptors.request.use(config => {
  const token = localStorage.getItem('token');
  if (token) {
    config.headers.Authorization = `Bearer ${token}`;
  }
  return config;
});

// 登录
async function login(username, password) {
  const response = await api.post('/auth/login', {
    username,
    password: md5(password)
  });
  return response.data;
}

// 查询历史记录
async function getHistory(params) {
  const response = await api.get('/history/operations', { params });
  return response.data;
}
```

#### cURL 示例

```bash
# 登录
curl -X POST http://localhost:8080/api/auth/login \

# 查询历史记录
curl -X GET "http://localhost:8080/api/history/operations?page=1&pageSize=20" \
  -H "Authorization: Bearer {token}"

# 导出数据
curl -X GET "http://localhost:8080/api/history/export?startTime=2026-03-01T00:00:00Z" \
  -H "Authorization: Bearer {token}" \
```
